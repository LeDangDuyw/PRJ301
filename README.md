# PRJ301 - HomeElectro

A Java Web-based e-commerce application for household electronics, developed as an academic project for PRJ301.

## Technologies
- Java Servlet
- JSP
- JSTL
- JDBC
- SQL Server
- Apache Tomcat
- NetBeans
- MVC Architecture

## Features
- User registration and login
- Product listing and product details
- Shopping cart
- Order placement
- Admin management
- SQL Server database integration

## How to Run
1. Import project into NetBeans
2. Configure SQL Server database using `PRJ.sql`
3. Deploy with Apache Tomcat
4. Run the application
